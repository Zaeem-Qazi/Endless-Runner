# 🏃 Endless Runner

A 2D endless runner game built in Unity using C#. The player must avoid obstacles, survive for as long as possible, and achieve the highest score.

## Technologies

- Unity
- C#
- TextMeshPro

## Features

- Endless procedural gameplay
- Jump-based movement system
- Dynamic obstacle spawning
- Score tracking
- Game Over and restart system

## Skills Demonstrated

- Object-oriented programming (OOP)
- Unity Physics
- Collision Detection
- Prefab Management
- UI Programming
- Game State Management
